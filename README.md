# Izmorozok Product Architecture

Public architecture map for **Izmorozok Systems**.

This repository shows how a Russian-language AI-assisted product system is structured: product layers, diagnostic logic, content workflow, and portfolio-safe documentation.

## Purpose

This repository demonstrates:

- product architecture
- diagnostic system structure
- content workflow design
- documentation logic
- public portfolio-safe project mapping

Commercial materials, internal prompts, paid product logic, and production diagnostics are not included.

## Русская версия
